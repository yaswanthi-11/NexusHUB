Backend

The FastAPI backend accepts text and file uploads, performs emergency keyword detection, and optionally calls an AI model when an API key is available.

Frontend

Safety flow

Text input is checked for red flags before AI engagement. File uploads are acknowledged while preserving the demo's educational disclaimer.
